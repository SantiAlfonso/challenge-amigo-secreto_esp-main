## Descripción del proyecto

Esta aplicación permite a un grupo de amigos organizar un sorteo de "Amigo Secreto". Los usuarios pueden agregar nombres a una lista y sortear aleatoriamente un amigo secreto.

## Estado del proyecto

🚀 Terminado
